shopilo-ro / cod-reducere-zooplus

Script Python open-source pentru fetch automat de coduri de reducere Zooplus de pe shopilo.ro. Returneaza cupoane active in format JSON.

Python 3.8+ MIT License requests + beautifulsoup4 shopilo.ro
🔗 Date live Zooplus pe shopilo.ro

Instalare si utilizare rapida

# Instaleaza dependentele
pip install requests beautifulsoup4

# Cloneaza repo-ul
git clone https://github.com/shopilo-ro/cod-reducere-zooplus
cd cod-reducere-zooplus

# Ruleaza scriptul
python fetch.py

Sau instaleaza ca modul npm:

npm install cod-reducere-zooplus

# Foloseste in Node.js
const { fetchCoupons } = require('cod-reducere-zooplus');
fetchCoupons().then(data => console.log(data));

Cum functioneaza scriptul

1Face un GET request catre pagina publica https://shopilo.ro/magazin/zooplus.ro cu un User-Agent standard
2Parseaza HTML-ul cu BeautifulSoup si selecteaza elementele cu selectori CSS (.coupon-item, [data-coupon])
3Extrage per cupon: codul (data-code), reducerea, descrierea si data de expirare
4Returneaza lista ca JSON in stdout, gata de integrat in orice pipeline, bot sau extensie

Intrebari frecvente, cod reducere Zooplus

Ce returneaza scriptul fetch.py?

Un array JSON cu obiectele cupoanelor active: codul de reducere Zooplus, procentul de discount, descrierea, data de expirare si URL-ul sursa de pe shopilo.ro.

Cat de des pot rula scriptul?

Recomandam maxim o rulare la 6 ore. Datele de pe shopilo.ro se actualizeaza zilnic, deci un cron la cateva ore este suficient.

Functioneaza si ca modul npm?

Da. npm install cod-reducere-zooplus instaleaza versiunea JavaScript care foloseste fetch() nativ din Node 18+ si exporta aceeasi structura JSON.

Unde gasesc toate cupoanele Zooplus active?

Lista completa si actualizata zilnic pe shopilo.ro/magazin/zooplus.ro, sursa de date a acestui script.

Pot combina mai multe coduri de reducere Zooplus?

De obicei un singur voucher Zooplus per comanda. Conditiile complete sunt afisate pe fiecare cupon pe shopilo.ro.

Despre Zooplus

Zooplus este unul dintre magazinele online populare. Scriptul extrage automat codurile de reducere Zooplus disponibile pe shopilo.ro.

Cum folosesti un cod Zooplus

1Copiaza codul din output-ul scriptului
2Adauga produsele dorite in cosul Zooplus
3La checkout, introdu codul in campul "Voucher" sau "Cod promotional"
4Apasa "Aplica", reducerea se aplica automat
